NAIApp 1.1.1 업데이트 소식

더 쉽게 업데이트하고, 더 편하게 사용할 수 있도록 다듬었습니다.
버전명이 일관성 있게 변경되었습니다. 이제부터는N(대형 업데이트).N(기능 추가).N(버그 수정)순으로 표시됩니다.

🚀 바로 느껴지는 변화

  • 앱에서 새 버전을 바로 알려드립니다. 알림 창에서 내용을 확인하고 곧바로 설치를 시작할 수 있습니다.
  • 다운로드가 끝나면 자동으로 설치 화면이 열려, 별도 파일 탐색 없이 최신 버전을 적용할 수 있습니다.
  • v1.0 버전까지는 자동업데이트 지원이 불가능합니다! 1.1.1로 변경한 이후부터 자동 업데이트 알림이 표시되니 이번 버전까지는 꼭 직접 업데이트 해주세요

✨ 편하게 쓸 수 있도록

  • 랜덤 시드 옵션을 켜면 시드 입력칸이 자동으로 잠기고, 끄면 깨끗하게 비워집니다.
  • 설정 패널의 접기/펼치기 버튼이 현재 상태를 바로 보여줘 원하는 영역을 빠르게 찾을 수 있습니다.

🛡️ 안정성과 안내

  • 앱 시작 시 필요한 초기 작업을 정리해 업데이트 알림이 끊김 없이 동작합니다.
  • 설치 권한 안내를 보완해 다운로드부터 설치까지 한 흐름으로 이어집니다.

🧑‍💻 개발자 메모

이제부터는 아카라이브에서 링크를 통해 업데이트를 다운받을 필요 없이 앱 내에서 바로 업데이트를 설치할 수 있습니다.

사용 중 불편한 점이나 아이디어가 있다면 언제든지 알려주세요. 다음 업데이트 준비에 큰 도움이 됩니다.
