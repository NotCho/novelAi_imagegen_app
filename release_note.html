NAIApp 1.3.99 BETA 업데이트 소식

와일드카드 시스템이 새롭게 추가되었습니다. 이번 버전은 BETA 버전으로, 안정성 확인이 필요할 수 있습니다.

[목록] 와일드카드 가중치 지원 (NAIA 호환)

  • 와일드카드 파일(.txt) 내에서 옵션: 가중치 형식을 지원합니다.
  • 가중치 숫자가 높을수록 해당 옵션이 선택될 확률이 높아집니다. (기본값 100)
  • 기존 NAIA 와일드카드 파일들을 그대로 가져와서 사용할 수 있습니다.
  • NAIA 와 가중치 방식이 다르다면 제보 부탁드립니다!!!

[검색] 더 직관적인 프롬프트 작성

  • 문법 하이라이팅: 프롬프트 입력창의 __wildcard__ 패턴이 시각적으로 강조됩니다.
  • 실시간 유효성 검사: 존재하는 와일드카드는 컬러로, 없는 와일드카드는 회색+취소선으로 표시됩니다.
  • 스마트 파싱: 존재하지 않거나 비활성화된 와일드카드는 생성 시 자동으로 제외되며, 불필요한 쉼표와 공백을 깔끔하게 정리해줍니다.

편의성 및 품질 개선

  • 원본 보존: 이제 와일드카드가 포함된 원본 프롬프트가 저장됩니다. 앱을 껐다 켜도 와일드카드가 유지되어 매번 수정할 필요가 없습니다.
  • 캐릭터 프롬프트 연동: 메인 프롬프트뿐만 아니라 캐릭터 프롬프트에서도 와일드카드를 자유롭게 사용할 수 있습니다.
  • 안정성: 특정 상황에서 하이라이트 기능이 오작동하여 앱이 멈추던 문제를 해결했습니다.


와일드카드 설정은 '설정' 탭에서 확인하실 수 있습니다.
